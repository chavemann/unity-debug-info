# DebugInfo

A simple utility package for displaying real time in-game debug data and notifications in Unity.

```csharp
DebugInfo.Log("A single line of text.");
DebugInfo.Log("Key", "Value");

DebugInfo.Notify("Hello World");
DebugInfo.NotifyEnabled("Debugging", IsDebuggingEnabled);
```

See the [Documentation](https://chavemann.github.io/unity-debug-info/) for a more complete overview.
